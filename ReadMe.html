Cobol To Json

This project converts Cobol Data Files to JSon. See The Description for details on use.

License: LGPL

RecordEditor Support

The RecordEditor can generate Cobol to Json scripts from a Cobol Copybook and optionally Cobol-data-file.

  • Select option Generate >> Java~JRecord code for Cobol
  • Select Generate >> Cobol ==> Json >> ...
